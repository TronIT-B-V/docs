title: "Email templates: customise outgoing emails"
description: "Customise the subject and body of every email TRONVoice sends to your clients, with support for Dutch, English, German, French, and Spanish."
---

TRONVoice sends automated emails to your clients when you send an invoice, a quote, a payment reminder, and more. You can customise the text of every email type, per administration and per language, using a rich-text editor with a live preview.

Email templates are configured separately for each administration. Go to **Settings → Huisstijl → Email templates** for the administration you want to configure.

## Template types

| Template | When it is sent |
|---|---|
| **Invoice to client** | When you send an invoice to a client |
| **Invoice to subcontractor** | When you send an invoice to a subcontractor |
| **Credit invoice to client** | When you send a credit note to a client |
| **Credit invoice to subcontractor** | When you send a credit note to a subcontractor |
| **Payment reminder** | When an automatic or manual payment reminder is sent |
| **Payment confirmation** | When an invoice is marked as paid |
| **Quote to client** | When you send a quote to a client |
| **SEPA mandate request** | When you invite a client to set up a direct debit mandate |
| **SEPA pre-notification** | Mandatory advance notice before collecting a direct debit payment |
| **Mandate confirmed** | Sent to you when a client confirms a direct debit mandate |

## Edit a template

<Steps>
  <Step title="Open email templates">
    Go to **Settings → Huisstijl** for the relevant administration, then click **Email templates bewerken**.
  </Step>
  <Step title="Select a template type">
    Click the template you want to edit from the list on the left.
  </Step>
  <Step title="Select a language">
    Use the language tabs at the top of the editor to switch between NL, EN, DE, FR, and ES. Each language has its own subject line and body text.
  </Step>
  <Step title="Edit the subject and body">
    Update the **subject** field and the **body** using the rich-text editor. Use variables (listed below) to insert dynamic content such as the invoice number or the client's name.
  </Step>
  <Step title="Preview">
    The live preview on the right shows exactly how the email will appear to your client, with example data substituted for the variables.
  </Step>
  <Step title="Save">
    Click **Save**. The custom template is used immediately for that template type and language. If no custom template exists for a language, TRONVoice falls back to the built-in default.
  </Step>
</Steps>

<Tip>
  You only need to customise the languages you actually use. If a client's language has no custom template, the default template for that language is used automatically.
</Tip>

## Multi-language support

TRONVoice supports five languages for outgoing emails:

| Code | Language |
|---|---|
| `nl` | Nederlands |
| `en` | English |
| `de` | Deutsch |
| `fr` | Français |
| `es` | Español |

The language used for a specific email is determined by the language set on the client's record. If no language is set on the client, the administration's default language is used.

## Available variables

Insert variables into your subject or body text using the `{variable}` syntax. TRONVoice replaces each variable with real data when the email is sent.

### Common variables (available in most templates)

| Variable | Description |
|---|---|
| `{bedrijfsnaam}` | Your company name (the administration) |
| `{klant_bedrijfsnaam}` | The client's company name |
| `{klant_contactpersoon}` | The client's contact person |
| `{klant_email}` | The client's email address |
| `{factuurnummer}` | Invoice number |
| `{factuurdatum}` | Invoice date |
| `{vervaldatum}` | Payment due date |
| `{totaal}` | Total amount including VAT |
| `{subtotaal}` | Subtotal excluding VAT |
| `{btw_totaal}` | VAT amount |
| `{omschrijving}` | Invoice description |
| `{iban}` | Your IBAN account number |
| `{iban_tnv}` | Account holder name |

## Live preview

The template editor includes a live preview panel that renders the email with realistic example data. The preview updates as you make changes to the subject or body, and you can switch the preview language using the language selector in the top-right corner of the preview panel.

<Note>
  The preview uses sample data, not actual client or invoice data. The final email your client receives will contain real values.
</Note>
