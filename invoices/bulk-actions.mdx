---
title: "Bulk actions for invoices"
description: "Select multiple invoices in TRONVoice and send, resend, mark paid, send reminders, delete, or download them in one operation."
---

When you work with many invoices at once, the **Invoices** list lets you select multiple invoices and apply an action to all of them in a single step. This saves time during month-end billing, debt collection, and bookkeeping.

## Select invoices

Each row in the invoice list has a checkbox. Tick the checkboxes next to the invoices you want to act on, or use **Select all** (**Selecteer alles**) at the top of the list to select every invoice on the current page.

As soon as one or more invoices are selected, a **bulk actions bar** appears showing the number of selected invoices and the available actions.

<Note>
Each bulk action only applies to invoices in a compatible status. Invoices that do not qualify are skipped automatically, and TRONVoice reports how many were skipped and why.
</Note>

## Available bulk actions

  <Card title="Resend" icon="repeat">

  </Card>
  <Card title="Mark as paid" icon="check">

  <Card title="Send reminders" icon="bell">

  <Card title="Delete (drafts)" icon="trash-2">

## Bulk send draft invoices

Use **Send (drafts)** (**Verzenden (concept)**) to send draft invoices for the first time.

<Steps>
  <Step title="Select draft invoices">
    Tick the checkboxes next to the drafts you want to send.
  </Step>
  <Step title="Open bulk actions">
    From the bulk actions bar, choose **Send (drafts)**.
  </Step>
  <Step title="Choose recipients and confirm">
    Select whether to send to **clients**, **admins**, or both, then confirm. TRONVoice processes each invoice in sequence, assigns invoice numbers, and deducts a credit per invoice.
  </Step>
</Steps>

Invoices are skipped with a warning when:
- The administration has no **IBAN** configured.
- Today is an **excluded invoicing day** for the client.
- You have insufficient credits for the number of invoices selected — TRONVoice redirects you to top up.

## Bulk resend invoices

Use **Resend** (**Opnieuw verzenden**) to send a fresh copy of invoices that are already **Sent** or **Paid**. Choose the recipients and confirm. Resending does not change the invoice status or number, and does not deduct extra credits.

## Bulk mark as paid

Use **Mark as paid** (**Markeer als betaald**) to record payment on multiple sent invoices at once.

In the dialog you can optionally set:
- A **payment date** (**Betaald op**) applied to all selected invoices — defaults to today.
- A **payment reference** (**Betaal referentie**) applied to all selected invoices.

Only invoices with the **Verzonden** status are marked as paid; others are skipped. If payment confirmation emails are enabled for the administration, a confirmation is sent to each client.

## Bulk send reminders

Use **Send reminders** (**Verstuur herinnering**) to send the next reminder step for each selected overdue invoice. TRONVoice automatically skips invoices that cannot receive a reminder — for example, those without a client email address, with the wrong status, or excluded from reminders.

## Bulk delete draft invoices

Use **Delete (drafts)** (**Verwijderen (concept)**) to permanently remove selected draft invoices and their line items.

<Warning>
Only **draft** invoices can be deleted. Sent, paid, or credited invoices are skipped — they have a permanent invoice number and cannot be removed. Deletion cannot be undone.
</Warning>

## Bulk download

Use **Download** to export the selected invoices as PDFs.

- If you select **one** invoice, it downloads directly as a PDF.
- If you select **multiple** invoices, TRONVoice bundles all PDFs into a single ZIP file named `Facturen-YYYY-MM-DD.zip`.

When an invoice has the **merge attachments** option enabled, its merged PDF (invoice plus PDF attachments) is included instead of the plain invoice.
